项目简介

HermesPortable 是 Hermes Agent（Nous Research 开发）的完全自包含便携版。它将 Python 运行时、依赖库、Node.js、配置面板和全部源代码打包到一个目录中，可直接从 U 盘在 macOS / Linux / Windows（WSL2）上运行，无需安装。

主要特性

功能说明
交互式 CLI多行编辑、流式输出、斜杠命令
消息网关Telegram / Discord / Slack / WhatsApp / Signal / Email
持久记忆MEMORY.md + USER.md 自动管理上下文
内置工具40+ 工具：终端、浏览器、搜索、文件操作、TTS、定时任务等
LLM 供应商支持 15+ 供应商（OpenRouter、Anthropic、OpenAI、Google、DeepSeek 等）
子代理子代理委托与并行工作流
终端后端本地 / Docker / SSH / Singularity / Modal / Daytona

构建便携包

构建需要 Python 3.8+、git、curl。

macOS / Linux

# 构建到默认输出目录 dist/HermesPortable/
python3 build.py

# 直接构建到 U 盘
python3 build.py /Volumes/MyUSB

Windows（在 WSL2 中）

git clone https://github.com/yuluyangguang1/hermes-portable.git
cd hermes-portable
python3 build.py
🚨
Windows 用户首次构建前，必须先完成 WSL2 环境配置。WSL2 是 Windows 子系统，Hermes 依赖它运行 Linux 环境。以下是完整配置步骤：

Windows 首次构建完整指南

1 启用 WSL2
以管理员身份打开 PowerShell，运行： 
wsl --install
该命令自动启用 WSL、安装 Linux 内核并设置 WSL2 为默认版本。安装完成后重启电脑。
2 安装 Ubuntu 发行版
重启后，系统会自动启动 Ubuntu 安装（如未启动，在 Microsoft Store 搜索 "Ubuntu" 安装）。首次打开需要设置 Linux 用户名和密码。
3 安装依赖工具
打开 Ubuntu 终端，依次运行： 
sudo apt update
sudo apt install -y python3 python3-pip git curl
4 克隆项目并构建
git clone https://github.com/yuluyangguang1/hermes-portable.git
cd hermes-portable
python3 build.py
构建默认输出到 dist/HermesPortable/。
5 从 Windows 访问构建产物
WSL2 中的文件可以通过文件管理器访问。在 Ubuntu 终端运行： 
explorer.exe dist/HermesPortable
这会直接在 Windows 资源管理器中打开构建目录。也可以将 dist/HermesPortable 复制到 U 盘或任意 Windows 目录。
6 启动运行
在 Windows 资源管理器中双击 Hermes.bat 即可启动（会自动调用 WSL2）。
📍

⚠
常见问题
  • wsl --install 报错 → 先运行 wsl --update 确保内核为最新版
  • WSL2 中网络慢 → 检查 VPN 是否影响 WSL2 代理，可尝试 wsl --shutdown 后重启
  • 磁盘空间不足 → WSL2 虚拟磁盘默认上限 256GB，运行 wsl --system -d Ubuntu df -h 查看使用情况
  • 构建中途断开 → WSL2 终端窗口关闭会终止进程，建议在 Windows Terminal 中运行
✅
构建产物约 500MB，包含：独立 CPython 3.12、Node.js v23、hermes-agent 全部依赖、Web UI。

免费模型配置

以下方案均无需付费即可使用 LLM，适合体验和轻度使用。按推荐程度排序。

方案一：OpenRouter（推荐）

注册即可获取免费 API Key，支持多个免费模型。

1前往 openrouter.ai/keys 注册并获取 API Key
2编辑便携包 data/.env 文件，添加：
OPENROUTER_API_KEY=sk-or-v1-你的密钥
3在配置面板或通过命令设置模型：
hermes config set model.default "openrouter/elephant-alpha"
免费模型 ID说明
openrouter/elephant-alphaOpenRouter 自有模型，免费
openrouter/owl-alphaOpenRouter 自有模型，免费
tencent/hy3-preview:free腾讯 HY3 免费版
minimax/minimax-m2.5:freeMiniMax M2.5 免费版
nvidia/nemotron-3-super-120b-a12b:freeNVIDIA Nemotron 免费版
arcee-ai/trinity-large-preview:freeArcee Trinity 免费版
💡
OpenRouter 会动态上线新的免费模型。在 Hermes 中运行 hermes model，选择 OpenRouter 后可浏览所有当前可用的免费模型。

方案二：Google Gemini 免费层

Google 提供免费 API 额度，但有速率限制（可能触发 429 错误）。

方式 A：API Key

1前往 Google AI Studio 创建免费 API Key
2编辑 data/.env，添加：
GOOGLE_API_KEY=你的密钥
3hermes config set model.default "gemini/gemini-3-flash-preview"

方式 B：OAuth 免密钥

1运行 hermes model，选择 google-gemini-cli 供应商
2按提示完成浏览器 OAuth 授权即可，无需 API Key
可用模型说明
gemini-3.1-pro-preview最强，免费层有限额
gemini-3-pro-preview次强
gemini-3-flash-preview速度快，推荐日常使用
gemini-3.1-flash-lite-preview最轻量

方案三：Nous Portal 免费层

Hermes 官方提供的免费模型入口，通过 OAuth 登录。

1运行 hermes login 或 hermes model，通过浏览器完成 OAuth 登录
2系统自动检测免费层，仅显示可用的免费模型
3选择模型后即可使用，无需手动配置 API Key
⚠
限制：免费层用户无法使用消息网关（Telegram / Discord 等机器人功能），仅支持 CLI 对话。

已知免费模型：xiaomi/mimo-v2-pro、minimax/minimax-m2.5:free 等。

方案四：Hugging Face 推理服务

Hugging Face 提供每月 $0.10 的免费额度，可调用多种开源模型。

1前往 huggingface.co/settings/tokens 创建 Token（勾选 "Make calls to Inference Providers"）
2编辑 data/.env，添加：
HF_TOKEN=hf_你的令牌
3hermes config set model.default "huggingface/deepseek-ai/DeepSeek-V3.2"
可用模型类型
deepseek-ai/DeepSeek-V3.2通用大模型
Qwen/Qwen3.5-397B-A17B超大参数模型
Qwen/Qwen3.5-35B-A3B轻量高效
moonshotai/Kimi-K2.5Kimi 系列
XiaomiMiMo/MiMo-V2-Flash小米 MiMo
MiniMaxAI/MiniMax-M2.5MiniMax
zai-org/GLM-5智谱 GLM

方案五：Qwen OAuth（通义千问）

通过 Qwen CLI 登录后免密钥使用。

1先安装 Qwen CLI 并运行 qwen auth qwen-oauth 完成登录
2hermes config set model.default "qwen-oauth/qwen-max"

方案六：本地部署（完全免费）

在本机运行模型，无需任何 API Key，适合有显卡的用户。

工具配置说明
Ollamaprovider: "ollama"最易上手，ollama pull 拉模型即可
LM Studioprovider: "lmstudio"图形界面，本地运行 http://127.0.0.1:1234
vLLMprovider: "vllm"高性能推理，适合生产环境
llama.cppprovider: "llamacpp"轻量级 C++ 推理引擎
 
# 以 Ollama 为例
# 1. 安装 Ollama 后拉取模型
ollama pull llama3.1

# 2. 配置 Hermes 使用 Ollama
hermes config set model.default "ollama/llama3.1"
🎯
方案选择建议：
  • 零门槛：OpenRouter（注册即用）或 Nous Portal（OAuth 登录）
  • 高质量：Google Gemini 免费层（模型最强）
  • 多样化：Hugging Face（可选模型最多）
  • 完全离线：Ollama 本地部署（无网络也可用）

常见问题

启动后浏览器没有自动打开？
手动访问 http://127.0.0.1:17520，或运行 hermes setup 进入配置向导。
Linux 上提示 Python 不兼容？
便携包在 macOS 构建的 Python 无法在 Linux 上运行。请确保系统安装了 Python 3.12，然后运行 ./linux-rebuild.sh 重建。
Windows 上无法运行？
Windows 支持需要先安装 WSL2（Windows Subsystem for Linux），然后在 WSL2 终端中运行。
API Key 配置在哪？
编辑 data/.env 文件，填入对应供应商的 Key，如 ANTHROPIC_API_KEY=sk-xxx。
如何诊断问题？
运行 hermes doctor 自动诊断，或查看 data/logs/ 下的日志文件。
