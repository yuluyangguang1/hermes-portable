📦 第一步：安装必要工具

构建前需要安装两个工具：Python（编程语言）和 Git（下载代码用）。

如果显示 Python 3.x.x 说明已安装，跳到第二步。

如果提示 command not found，继续下面的步骤。

点击大的黄色按钮 "Download Python 3.12.x"

3安装 Git

在终端输入：

git --version

如果提示安装 Xcode Command Line Tools，点 "安装" 等待完成即可。

💾 直接构建到U盘（可选）

如果你想直接把产物构建到U盘，而不是先构建再复制：

macOS / Linux

python3 build.py /Volumes/你的U盘名

Windows

先用资源管理器打开U盘，在地址栏输入 cmd 回车，然后：

python build_windows.py

构建产物会直接生成在U盘的 dist\HermesPortable\ 中。

🚀 第四步：首次使用

1双击启动

系统双击的文件
macOSHermes.command
WindowsHermes.bat
Linux终端运行 ./Hermes.sh

2配置 API Key

首次启动会自动打开浏览器，显示配置面板。

选择一个 AI 提供商（推荐 OpenRouter），填入你的 API Key，点击 "测试连接"，成功后点击 "启动"。

3开始对话

启动后终端中即可与 AI 对话。所有数据存储在U盘的 data/ 目录中。

❓ 常见问题

Q: 构建时提示 "git clone failed" 怎么办？

A: 网络问题。确保能访问 GitHub。如果在中国大陆，可能需要使用代理。

Q: Windows 双击 .bat 没反应？

A: 右键 Hermes.bat → 以管理员身份运行。如果还是没反应，打开 CMD，手动输入 Hermes.bat 查看错误信息。

Q: macOS 提示 "无法打开，因为无法验证开发者"？

A: 右键 Hermes.command → 打开，然后点 "仍然打开"。

Q: 构建时提示 "No module named pip"？

A: Python 安装不完整。重新安装 Python，确保安装时勾选了 pip。

Q: 可以不用构建，直接下载使用吗？

A: 可以！从 GitHub Releases 下载预构建包，解压即可使用。
